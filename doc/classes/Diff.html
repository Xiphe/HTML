      getA(int $start, int $end) : array

      If no values are supplied, the entire string is returned. It's also possible to specify just one line to return only that line.

      Parameters

      int

      The starting number.

      $end

      The ending number. If not supplied, only the item in $start will be returned.

      Generate a list of the compiled and grouped opcodes for the differences between the two strings.

      getGroupedOpcodes() : array

      Generally called by the renderer, this class instantiates the sequence matcher and performs the actual diff generation and return an array of the opcodes for it. Once generated, the results are cached in the diff class instance.

      Returns

      arrayArray of the grouped opcodes for the generated diff.
